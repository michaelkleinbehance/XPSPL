Synopsis

Prggmr is a high-performance event processor, it uses Signals, Subscriptions, Queues and an Engine, that have been engineered from the ground up to be incredibly lightweight, fast, efficent and robust with simplicity in mind.

Prggmr is much more than an event library, it is a robust standalone event processing engine written in PHP, allowing for application & server level php programming as if it wasn't even php.

Within prggmr you can

subscribe

subscribe(function(){
    echo "HelloWorld";
}, "helloworld");

fire

fire("helloworld");

chain

chain("yourfunction", "helloworld");

setInterval

setInterval(function(){
    echo "HelloWorld";
}, 1000);

setTimeout

setTimeout(function(){
    echo "5 seconds just passed.";
}, 5000);

without the need for some complex configuration or monolithic library.
