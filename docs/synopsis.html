Synopsis

prggmr is a high-performance event processing engine, it uses Events, Signals, Subscriptions, Queues and an Engine, that have been engineered from the ground up to be incredibly lightweight, fast, efficent and robust with simplicity and ease of use in mind.

prggmr is much more than an event library, it is a robust standalone event processing engine written in PHP, allowing for application & server level php programming as if it wasn't even php.

prggmr allows you to

subscribe

subscribe(function(){
    echo "HelloWorld";
}, "helloworld");

fire

fire("helloworld");

chain

chain("yourfunction", "helloworld");

setInterval

setInterval(function(){
    echo "HelloWorld";
}, 1000);

setTimeout

setTimeout(function(){
    echo "5 seconds just passed.";
}, 5000);

with no configuration, no autoloading and no monolithic library, you need "just a single line of code." 

include 'lib/prggmr.php';
